# Contributing to ASIF

We welcome research, code, and documentation contributions!

## How to Contribute
- Fork the repository and create a feature branch
- Add or update documentation in `docs/`
- Submit code examples or protocol improvements
- Open a pull request with a clear description

## Areas for Contribution
- Protocol extensions (new endpoints, schemas)
- LLM agent implementations (Python, JS, etc.)
- Marketplace registry and reputation algorithms
- Security and sandboxing improvements
- Real-world service provider adapters

## Research Topics
- LLM tool-use reliability
- Decentralized/federated marketplace
- Confidential computing for agents
- Policy learning for OPA

---

For questions, open an issue or join the discussion board.
